# 📡 Data Communications

> 데이터 통신 및 통신이론의 기초 개념을 학습하고 정리하는 공간입니다.

이 디렉터리는 유튜브 강의와 **『기초통신이론 : 디지털 통신 중심으로』** 교재를 기반으로 통신이론을 학습하고, 핵심 개념을 이해한 방식으로 정리하기 위해 작성되었습니다.

# 🎯 Learning Objectives

* 통신 시스템의 기본 개념 이해
* 신호와 시스템 기초
* 아날로그 통신 방식 이해
* 디지털 통신 원리 이해
* 변조 및 복조 방식 이해
* 정보이론 및 부호이론 기초 학습
* 통신 시스템 성능 분석
