# 📖 Knowledge Base

> 네트워크 엔지니어로 성장하기 위해 학습한 이론과 기술을 체계적으로 정리하는 공간입니다.

이 디렉터리는 자격증 학습, 컴퓨터 공학 기초, 네트워크 이론 및 운영에 필요한 기술 문서를 기록하기 위해 만들어졌습니다.

---

# 🎯 Purpose

* 기초 이론 정리 및 복습
* 자격증 학습 내용 기록
* 실무에서 필요한 기술 개념 정리
* 실습(Labs) 및 운영(Maintenance)의 이론적 배경 문서화
* 지속적인 기술 학습 아카이브 구축
