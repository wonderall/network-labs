# 🛠️ Maintenance Operations

> 네트워크 유지보수 업무 중 실제 수행한 점검 및 운영 절차를 정리한 문서입니다.

본 저장소는 장애 대응이 아닌 **운영 환경에서 수행한 점검, 상태 확인, 유지보수 절차 및 작업 기록**을 관리합니다.

---

## 주요 업무

### Network Health Check
- Interface 상태 확인
- VLAN 구성 확인
- Trunk 상태 확인
- STP 상태 확인
- MAC Address Table 확인
- ARP Table 확인
- Routing Table 확인

### System Health Check
- CPU 사용률 확인
- Memory 사용률 확인
- Temperature 확인
- Fan / Power 상태 확인
- Log 확인

### Configuration Review
- Running Configuration 확인
- Startup Configuration 확인
- Backup 여부 확인
- 변경사항 확인

### Verification
- Ping Test
- Gateway 확인
- Interface Error 확인
- Link Status 확인

---

## Purpose

- 운영 절차 문서화
- 점검 항목 정리
- 유지보수 업무 기록
- 장비별 확인 명령어 정리

---

> 실제 운영 환경에서 수행한 점검 및 확인 업무를 기반으로 작성하였으며,
> 구성 변경이나 신규 구축을 수행하지 않은 항목은 포함하지 않습니다.
