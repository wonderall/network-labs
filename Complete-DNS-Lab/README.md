# 📖 About

DNS(Domain Name System)는 인터넷에서 가장 중요한 서비스 중 하나입니다.

웹 브라우저, 이메일, 클라우드, Active Directory, CDN, Load Balancer…

이 모든 기술은 DNS 위에서 동작합니다.

하지만 대부분의 실습 자료는

"DNS는 도메인을 IP로 변환한다."

라는 한 줄 설명으로 끝나는 경우가 많습니다.

이 프로젝트는 그런 부족한 부분을 채우기 위해 만들어졌습니다.

본 프로젝트에서는

- DNS 기초
- DNS Query
- DNS Cache
- DNS Record
- Packet Tracer 실습
- EVE-NG BIND9 구축
- Enterprise DNS
- DNS Security
- Route53

까지 단계적으로 학습합니다.
