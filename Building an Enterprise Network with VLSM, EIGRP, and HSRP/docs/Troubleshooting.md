# 🔧 Troubleshooting

## EIGRP Neighbor Not Formed

Check:

- AS Number
- Network Command
- Interface Status
- IP Address

---

## No Connectivity

Check:

- Interface Status
- Routing Table
- Default Gateway
- IP Address

---

## HSRP Not Working

Check:

- Virtual IP
- Group Number
- Priority
- Interface Status
