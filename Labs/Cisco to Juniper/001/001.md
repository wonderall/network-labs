# Juniper vLab - Basic Device Configuration

## 목표
- 두 대의 Juniper 라우터(vSRX/vMX 등)를 연결
- Hostname 설정
- Root Password 설정
- Password Hash 동작 확인

> **참고**
>
> Juniper Junos OS에는 Cisco의 `enable password` 또는 `service password-encryption`과 같은 기능이 없습니다.
> 모든 사용자 비밀번호는 **자동으로 Hash(SHA-512 등)** 되어 저장됩니다.

---

# 1. R1과 R2를 GigabitEthernet 인터페이스로 연결

### 연결 방식

```text
R1 ge-0/0/0
      │
      │ Ethernet Cable
      │
R2 ge-0/0/0
```

(vLab 환경에서는 연결만 확인하면 됩니다.)

---

# 2. Hostname 설정

## R1

```bash
configure
set system host-name R1
commit
```

## R2

```bash
configure
set system host-name R2
commit
```

# 3. Root Password를 'cisco'로 설정

## R1

```bash
configure

```

입력 예시

```text
New password: cisco
Retype new password: cisco
```

Commit

```bash
commit
```

# 4. Configuration에서 Password 확인

명령어

```bash
show configuration system
```

출력 예시

```text
system {
    root-authentication {
        encrypted-password "$6$....";
    }
}
```

### 결과

✅ Password는 Hash 형태로 저장됨

---

# 5. Password Encryption 활성화

Juniper에서는 별도의 설정이 없습니다.

비밀번호를 입력하는 순간 자동으로 Hash 처리됩니다.

```text
별도의 명령어 없음
```

---

# 6. Configuration에서 Password 확인

명령어

```bash
show configuration system
```

출력 예시

```text
encrypted-password "$6$G4uM......"
```

### 결과

✅ 항상 암호화(Hash)되어 저장됨

---

# 7. Password Encryption 비활성화

Juniper에서는 지원하지 않습니다.

```text
불가능
```

모든 Password는 항상 Hash 형태로 저장됩니다.

# Cisco와 Juniper 비교

| 기능 | Cisco IOS | Juniper Junos |
|------|------------|---------------|
| Hostname 설정 | `hostname R1` | `set system host-name R1` |
| 관리자 암호 | `enable password` | `root-authentication` |
| Password Encryption | `service password-encryption` 필요 | 항상 자동 적용 |
| 평문 저장 가능 | 가능 | 불가능 |
| 암호 저장 방식 | Plain Text 또는 Type 7 | SHA 기반 Hash |
| 설정 저장 | `copy running-config startup-config` | `commit` |

---

# 학습 포인트

- Juniper는 Cisco의 `enable password` 개념이 없으며, 대신 **root 사용자 인증**을 사용한다.
- `plain-text-password`로 입력해도 실제 설정 파일에는 **자동으로 Hash** 되어 저장된다.
- Cisco의 `service password-encryption`처럼 암호화를 활성화하거나 비활성화하는 기능은 존재하지 않는다.
- Junos는 기본적으로 보안을 고려하여 **모든 사용자 비밀번호를 암호화(Hash)하여 저장**한다.
